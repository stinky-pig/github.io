Git

Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

Git 与 SVN 区别

  • Git 是分布式的，SVN 不是
  • Git 把内容按元数据方式存储，而 SVN 是按文件
  • Git 分支和 SVN 的分支不同
  • Git 没有一个全局的版本号，而 SVN 有
  • Git 的内容完整性要优于 SVN
